# FreeRTOS Multitasking Demo  
Simulated an RTOS system on Wokwi with **task scheduling** and **Serial output**.  

## Features  
- **Task 1**: Blinks LED every 500ms.  
- **Task 2**: Prints "Hello from FreeRTOS!" every 1s.
